Extract Links From A HTML File With PHP

By philipnorton42 on 6th March 2008

Use the following function to extract all of the links from a HTML string.

  1. function linkExtractor($html)
  2. {
  3.  $linkArray = array();
  4.  if(preg_match_all('/]*)[\"\']?[^>]*>(.*?)/i', $html, $matches, PREG_SET_ORDER)){
  5.   foreach ($matches as $match) {
  6.    array_push($linkArray, array($match[1], $match[2]));
  7.   }
  8.  }
  9.  return $linkArray;
  10. }

To use it just read a web page or file into a string, and pass that string to the function. The following example reads a web page using the PHP CURL functions and then passes the result into the function to retrieve the links.

  1. $url = 'http://www.hashbangcode.com';	
  2. $ch = curl_init();
  3. curl_setopt($ch,CURLOPT_URL,$url);
  4. curl_setopt($ch,CURLOPT_USERAGENT, 'Mozilla/5.0 (Windows; U; Windows NT 5.1; en-GB; rv:1.8.1.12) Gecko/20080201 Firefox/2.0.0.12');
  5. curl_setopt($ch,CURLOPT_HEADER,0);
  6. curl_setopt($ch,CURLOPT_RETURNTRANSFER,1);
  7. curl_setopt($ch,CURLOPT_FOLLOWLOCATION,0);
  8. curl_setopt($ch,CURLOPT_TIMEOUT,120);
  9. $html = curl_exec($ch);
  10. curl_close($ch);
  11. echo '' . print_r(linkExtractor($html), true) . '<pre>';

The function will return an array, with each element being an array containing the link location and the text that the link contains.
